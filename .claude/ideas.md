# Pipeline para o futuro

- Antes de analisar as imagens, o agente deve coletar as novidades dos documentos da semana. Com isso, ele deve detectar oque existe de "novidade" na obra. Com essas novidades, ele deve adicionar ao prompt intruções como: "Verifique na imagem 2 se as seguintes informações são compatítveis: X, Y, Z (...)". O Objetivo é cruzar informações entre oque foi dado pela empreiteira e oque foi registrado nas fotos. 

- Para melhorar a precisão das respostas da IA, deve se utilizar informações de projeto para aumentar o contexto dela, já que não estamos trabalhando com um modelo especializado.
